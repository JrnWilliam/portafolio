Sobre Mi

Hola, Soy William Jirón tengo experiencia y habilidades en desarrollo web, análisis de datos y base de datos. Además, tengo habilidades avanzadas en SQL para consultar y manipular datos en bases de datos relacionales como SQL Server y PostgreSQL.
